# Public Assets

## Directory Structure

- `/images` - Static images (logo, placeholder images)
- `/ads` - Ad creative assets

## Add Your Assets

1. Logo: Replace `logo.svg` with your actual logo
2. Favicon: Add `favicon.ico` in this directory
3. Social sharing image: Add `og-image.png` (1200x630px recommended)

## WordPress Images

All article images are served from WordPress via Next.js Image Optimization.
Configure the WordPress domain in `next.config.mjs`.
